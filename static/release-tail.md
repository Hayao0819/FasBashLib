
## Load in script with curl

### UpperCase Version

```bash
# shellcheck source=/dev/null
source <(curl -sL "%SOURCEURL%")
```

### LowerCase Version

```bash
# shellcheck source=/dev/null
source <(curl -sL "%LOWERSOURCEURL%")
```

### SnakeCase Version

```bash
# shellcheck source=/dev/null
source <(curl -sL "%SNAKESOURCEURL%")
```
